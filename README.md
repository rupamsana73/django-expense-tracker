# Django Expense Tracker 

A web-based expense tracker application built using Django to manage income and expenses efficiently.

##  Features
- User authentication (Login & Signup)
- Add, edit, and delete income and expenses
- Dashboard with income vs expense overview
- Monthly transaction tracking
- Export transactions to CSV
- Secure user-based data handling

##  Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- SQLite (default database)
